# Lesman Ghazaryan — Academic Website

Static academic website for **Lesman Ghazaryan**, built with React, TypeScript, Tailwind CSS, and Vite.

## Important files

- `public/lesman-ghazaryan-photo.jpg` — exact original portrait
- `public/lesman-ghazaryan-cv.pdf` — exact original CV PDF
- `src/data.ts` — publications, research projects, teaching, awards, and profile information
- `.github/workflows/deploy-pages.yml` — automatic free deployment to GitHub Pages

The site contains no Gemini API calls, API keys, authentication, database, server, image generation, or CV generation.

## Local preview (optional)

```bash
npm install
npm run dev
```
